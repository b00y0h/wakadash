<task type="auto">
  <name>Task 2: Add homebrew_casks to GoReleaser config</name>
  <files>/workspace/wakadash/.goreleaser.yaml</files>
  <action>
    Add `homebrew_casks` section to .goreleaser.yaml after the `checksum` section:

    ```yaml
    homebrew_casks:
      - name: wakadash

        # Tap repository configuration
        repository:
          owner: b00y0h
          name: homebrew-wakadash
          token: "{{ .Env.HOMEBREW_TAP_TOKEN }}"

        # Cask metadata
        homepage: https://github.com/b00y0h/wakadash
        description: "Live terminal dashboard for WakaTime coding stats"
        license: MIT

        # Cask file location
        directory: Casks

        # Git commit configuration
        commit_author:
          name: goreleaserbot
          email: bot@goreleaser.com
        commit_msg_template: "Brew cask update for {{ .ProjectName }} version {{ .Tag }}"

        # macOS quarantine removal for unsigned binaries
        hooks:
          post:
            install: |
              if OS.mac?
                system_command "/usr/bin/xattr", args: ["-dr", "com.apple.quarantine", "#{staged_path}/wakadash"]
              end

        # Skip pre-release versions
        skip_upload: auto
    ```

    Key requirements:
    - Repository name is `homebrew-wakadash` (enables `brew tap b00y0h/wakadash` short form)
    - Token uses env var `HOMEBREW_TAP_TOKEN` (will be added to workflow)
    - Quarantine removal hook prevents "damaged application" errors on macOS
  </action>
  <verify>
    `goreleaser check --config /workspace/wakadash/.goreleaser.yaml` shows no errors.
    Grep confirms homebrew_casks section exists.
  </verify>
  <done>

  </done>

  <action>Create Personal Access Token for tap publishing</action>
  <instructions>
    GoReleaser needs a Personal Access Token to push casks to the homebrew-wakadash repository.
    The default GITHUB_TOKEN cannot write to other repositories.

    **Steps:**

    1. Go to GitHub Settings -> Developer settings -> Personal access tokens -> Fine-grained tokens
       URL: https://github.com/settings/personal-access-tokens/new

    2. Configure the token:
       - **Token name:** `wakadash-homebrew-tap`
       - **Expiration:** 1 year (set calendar reminder to renew in Feb 2027)

       - **Permissions:** Contents (Read and write)

    3. Generate token and copy the value (you won't see it again!)

    4. Add as repository secret:
       ```bash
       gh secret set HOMEBREW_TAP_TOKEN -R b00y0h/wakadash
       # Paste the token value when prompted
       ```

    5. Verify secret was added:
       ```bash
       gh secret list -R b00y0h/wakadash
       # Should show HOMEBREW_TAP_TOKEN in the list
       ```
