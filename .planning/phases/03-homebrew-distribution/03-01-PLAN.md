  <name>Task 1: Update GoReleaser to use homebrew_casks</name>
  <files>/workspace/wakafetch/.goreleaser.yaml</files>
  <action>
Replace the deprecated `brews` section with `homebrew_casks` section in .goreleaser.yaml.

Key changes from existing config:
1. Rename `brews:` to `homebrew_casks:`
2. Remove `test:` block (casks don't support test blocks - see research)
3. Remove `install:` block (casks use different installation mechanism)
4. Add `hooks:` section with post-install quarantine removal for macOS:
   ```yaml
   hooks:
     post:
       install: |
         if OS.mac?
           system_command "/usr/bin/xattr", args: ["-dr", "com.apple.quarantine", "#{staged_path}/wakafetch"]
         end
   ```
5. Add commit author and message template:
   ```yaml
   commit_author:
     name: goreleaserbot
     email: bot@goreleaser.com
   commit_msg_template: "Brew formula update for {{ .ProjectName }} version {{ .Tag }}"
   ```
6. Keep existing repository, directory, homepage, description, and license fields unchanged

Final homebrew_casks section should look like:
```yaml
homebrew_casks:
  - name: wakafetch
    repository:
      owner: b00y0h
      name: homebrew-wakafetch
      token: "{{ .Env.HOMEBREW_TAP_TOKEN }}"
    directory: Casks
    homepage: https://github.com/b00y0h/wakafetch
    description: "WakaTime activity fetcher CLI"
    license: MIT
    commit_author:
      name: goreleaserbot
      email: bot@goreleaser.com
    commit_msg_template: "Brew formula update for {{ .ProjectName }} version {{ .Tag }}"
    hooks:
      post:
        install: |
          if OS.mac?
            system_command "/usr/bin/xattr", args: ["-dr", "com.apple.quarantine", "#{staged_path}/wakafetch"]
          end
```
  </action>
