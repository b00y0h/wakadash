   ```bash

   brew untap b00y0h/wakafetch 2>/dev/null || true
   ```

   ```bash
   brew tap b00y0h/wakafetch
   ```

   ```bash
   brew install wakafetch
   ```

   ```bash
   which wakafetch
   ```
   Expected: Path like /usr/local/bin/wakafetch or /opt/homebrew/bin/wakafetch

   ```bash
   brew install b00y0h/wakafetch/wakafetch
   ```
