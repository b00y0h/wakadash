  <action>
Create internal/theme/config.go with functions to read/write theme from ~/.wakatime.cfg:

1. **LoadThemeFromConfig() (string, error)**
   - Get config path using os.UserHomeDir() + "/.wakatime.cfg"
   - Open file, scan line by line
   - Skip comments (# or ;) and blank lines
   - Parse key=value pairs, look for "theme" key
   - Return theme name if found, empty string if not found (first run)
   - Handle file not existing gracefully (return "", nil)

2. **SaveThemeToConfig(themeName string) error**
   - Read entire ~/.wakatime.cfg file
   - Find existing "theme = ..." line and update it
   - If not found, append "theme = {themeName}" to end
   - Write file back with mode 0600 (user read/write only)
   - Ensure file ends with newline

Follow same pattern as existing config/config.go but only for theme key.

Include helper: configFilePath() string (same as config package)
  </action>
  <verify>go build ./internal/theme/... compiles without errors</verify>
  <done>LoadThemeFromConfig and SaveThemeToConfig functions implemented; theme persists to ~/.wakatime.cfg</done>
