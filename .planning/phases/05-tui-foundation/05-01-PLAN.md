<task type="auto">
  <name>Task 2: Create TUI package with model, messages, commands, and styles</name>
  <files>wakadash/internal/tui/model.go, wakadash/internal/tui/messages.go, wakadash/internal/tui/commands.go, wakadash/internal/tui/styles.go</files>
  <action>
Create `wakadash/internal/tui/` directory and files following research patterns.

**messages.go** - Custom tea.Msg types:
- `statsFetchedMsg` with `stats *types.StatsResponse`
- `fetchErrMsg` with `err error`

**commands.go** - tea.Cmd factories:
- `fetchStatsCmd(client *api.Client, rangeStr string) tea.Cmd` - wraps client.FetchStats in closure, returns statsFetchedMsg or fetchErrMsg. Include recover() to catch panics and convert to fetchErrMsg (per research pitfall #2).

**styles.go** - lipgloss styles:
- `borderStyle` with RoundedBorder, color 62
- `titleStyle` bold, color 205
- `dimStyle` color 241
- `errorStyle` color 196 (red)

**model.go** - tea.Model implementation:
```go
type Model struct {
    // Layout - CRITICAL: Initialize to 80x24 (research pitfall #1)
    width  int
    height int

    // Data
    stats     *types.StatsResponse
    loading   bool
    err       error
    lastFetch time.Time

    // Dependencies
    client   *api.Client
    rangeStr string

    // UI components
    spinner spinner.Model

    // State
    quitting bool
}
```

Constructor `NewModel(client *api.Client, rangeStr string) Model`:
- Initialize width=80, height=24 (safe defaults before WindowSizeMsg)
- Initialize spinner with spinner.Dot, style color 205
- Set rangeStr default to "last_7_days" if empty

`Init() tea.Cmd`:
- Return `tea.Batch(fetchStatsCmd(m.client, m.rangeStr), m.spinner.Tick)`

`Update(msg tea.Msg) (tea.Model, tea.Cmd)`:
- `tea.WindowSizeMsg`: Store width/height
- `tea.KeyMsg`: Handle "q" and "ctrl+c" to quit (set quitting=true, return tea.Quit)
- `statsFetchedMsg`: Set loading=false, store stats, clear err, set lastFetch
- `fetchErrMsg`: Set loading=false, store err
- `spinner.TickMsg`: Update spinner, return its cmd

`View() string`:
- If quitting: return empty string
- If loading: return spinner + "Fetching stats..."
- If err: return error message styled with errorStyle
- Otherwise: render stats panel with:
  - Title: "WakaTime Stats ({rangeStr})"
  - Total time: stats.Data.HumanReadableTotal
  - Daily average: stats.Data.HumanReadableDailyAverage
  - Top 5 languages with time
  - Top 5 projects with time
  - Status bar: "Last updated: {lastFetch.Format(15:04:05)}  q quit"

Use lipgloss.JoinVertical for layout. Account for borders (-2 from dimensions).
