Create `wakadash/internal/archive/fetcher.go` with:

1. **Fetcher struct** with fields:
   - `HistoryRepo string` (format: "owner/repo")
   - `httpCli *http.Client` (with 10s timeout, matching api/client.go pattern)

2. **New function** `New(historyRepo string) *Fetcher`
   - Returns nil if historyRepo is empty (graceful no-op)
   - Validates format "owner/repo" using strings.Count(historyRepo, "/") == 1
   - Returns nil if format invalid (defer error to fetch time per Phase 11 decision)

3. **FetchArchive function** `(f *Fetcher) FetchArchive(date string) (*types.DayData, error)`
   - If f is nil, return nil, nil (no history_repo configured - graceful no-op)
   - Build URL: `https://raw.githubusercontent.com/{owner}/{repo}/main/data/{date}.json`
   - Make GET request (no auth needed for public repos)
   - **Handle 404**: Return nil, nil (not an error - data simply doesn't exist)
   - **Handle other errors**: Return nil, wrapped error
   - **Handle 200**: Decode JSON into types.DayData and return

The archive JSON format matches WakaTime's summary response for a single day (types.DayData).

Use patterns from api/client.go:
- Same 10-second timeout
- Same error handling style for network issues
- Same JSON decoding pattern
