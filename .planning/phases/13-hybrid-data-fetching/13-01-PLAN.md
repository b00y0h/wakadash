<feature>
  <name>Date-based data source routing</name>
  <files>wakadash/internal/datasource/source.go, wakadash/internal/datasource/source_test.go</files>
  <behavior>
    The DataSource determines which backend to use based on date:

    **IsRecent(date) logic:**
    - Input: date string "YYYY-MM-DD"
    - Output: true if date is within 7 days of today, false otherwise
    - Edge case: today returns true
    - Edge case: 7 days ago returns true (boundary)
    - Edge case: 8 days ago returns false (boundary)

    **Fetch(date) logic:**
    - If IsRecent(date): Use API client's FetchSummary (filter to single day)
    - If not IsRecent(date): Use archive fetcher's FetchArchive
    - Both return *types.DayData

    **Return type normalization:**
    - API returns SummaryResponse.Data[] - extract single day matching date
    - Archive returns DayData directly
    - Both paths return (*types.DayData, error)

    Test cases:
    - IsRecent("today") -> true
    - IsRecent("7 days ago") -> true
    - IsRecent("8 days ago") -> false
    - Fetch(recent_date) calls API
    - Fetch(old_date) calls archive
    - Fetch with nil archive fetcher returns (nil, nil) for old dates
  </behavior>
  <implementation>
    1. Create wakadash/internal/datasource/ directory
    2. Create source.go with:
       - DataSource struct: api *api.Client, archive *archive.Fetcher
       - New(client, fetcher) constructor
       - IsRecent(date string) bool - compare with time.Now()
       - Fetch(date string) (*types.DayData, error) - route based on IsRecent
       - extractDay(summary *types.SummaryResponse, date string) - helper
    3. Create source_test.go with behavioral tests:
       - Test IsRecent with various dates
       - Test Fetch routing (mock/spy on calls)
       - Test nil archive fetcher handling
  </implementation>
</feature>
