**1. Add selectedDate field to Model struct:**

```go
// Date navigation
selectedDate string // Currently viewed date (YYYY-MM-DD format), empty = today
```

Add after the archiveData field. Empty string means "today" (live data).

**2. Initialize selectedDate in NewModel:**
```go
selectedDate: "", // Empty means today (live data)
```

**3. Add key handlers in Update() after the existing ChangeTheme handler:**

For PrevDay (left arrow):
```go
case key.Matches(msg, m.keys.PrevDay):
    // Calculate previous day
    currentDate := m.selectedDate
    if currentDate == "" {
        currentDate = time.Now().Format("2006-01-02")
    }
    parsed, err := time.Parse("2006-01-02", currentDate)
    if err != nil {
        return m, nil
    }
    m.selectedDate = parsed.AddDate(0, 0, -1).Format("2006-01-02")
    return m, fetchDataCmd(m.dataSource, m.selectedDate)
```

For NextDay (right arrow):
```go
case key.Matches(msg, m.keys.NextDay):
    currentDate := m.selectedDate
    if currentDate == "" {
        // Already at today, can't go forward
        return m, nil
    }
    parsed, err := time.Parse("2006-01-02", currentDate)
    if err != nil {
        return m, nil
    }
    nextDay := parsed.AddDate(0, 0, 1)
    today := time.Now().Format("2006-01-02")
    if nextDay.Format("2006-01-02") >= today {
        m.selectedDate = "" // Return to live today view
    } else {
        m.selectedDate = nextDay.Format("2006-01-02")
    }
    return m, fetchDataCmd(m.dataSource, nextDay.Format("2006-01-02"))
```

For Today (0 or Home):
```go
case key.Matches(msg, m.keys.Today):
    if m.selectedDate == "" {
        return m, nil // Already at today
    }
    m.selectedDate = ""
    return m, fetchDataCmd(m.dataSource, time.Now().Format("2006-01-02"))
```

**4. Update dataFetchedMsg handler** to track when viewing historical vs today:
The existing handler already stores data in m.archiveData, which is correct.

**Note:** Auto-refresh behavior will be handled in Phase 15 (DISP-02, DISP-03). For now, navigation triggers immediate data fetch for the selected date.
