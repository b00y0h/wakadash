<task type="auto">
  <name>Task 2: Create GitHub Actions release workflow</name>
  <files>.github/workflows/release.yml</files>
  <action>
Create `.github/workflows/release.yml` with the following structure:

```yaml
name: release

on:
  push:
    tags:
      - 'v*.*.*'

permissions:
  contents: write

jobs:
  goreleaser:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Set up Go
        uses: actions/setup-go@v5
        with:
          go-version: stable

      - name: Run GoReleaser
        uses: goreleaser/goreleaser-action@v6
        with:
          distribution: goreleaser
          version: "~> v2"
          args: release --clean
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          HOMEBREW_TAP_TOKEN: ${{ secrets.HOMEBREW_TAP_TOKEN }}
```

CRITICAL requirements:
- `fetch-depth: 0` for full git history (changelog generation)
- `permissions: contents: write` for release creation
- `HOMEBREW_TAP_TOKEN` passed via env for cross-repo tap publishing
- Trigger pattern `v*.*.*` for semantic version tags only
  </action>
  <verify>YAML is valid and workflow file exists at correct path</verify>
  <done>.github/workflows/release.yml exists with correct structure, permissions, and secrets</done>
</task>
