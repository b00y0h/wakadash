### E2E Flow Verification

#### Flow 1: User Views Historical Week
1. User presses left arrow (PrevWeek)
2. Cache checked (instant if cached)
3. DataSource.FindNonEmptyWeek searches backward
4. weekSearchResultMsg updates selectedWeekStart
5. fetchDataCmd routes to archive.FetchArchive
6. dataFetchedMsg stores archiveData
7. getActiveStatsData returns archiveData to all panels
8. Status bar shows `[HISTORICAL] [week range]`
9. Auto-refresh paused
10. Background prefetch loads previous week

**Status:** COMPLETE

#### Flow 2: User Returns to Today
1. User presses 0 or Home (Today)
2. selectedWeekStart reset to ""
3. showEndOfHistory cleared
4. fetchDataCmd routes to API
5. Panels render current week data
6. Auto-refresh resumes

**Status:** COMPLETE

#### Flow 3: User Navigates to End of History
1. User presses left repeatedly
2. Each navigation checks cache first
3. Reaches week with no data
4. showEndOfHistory flag set
5. Full-screen banner with "End of History"
6. Navigation hints displayed
7. User presses right or 0
8. Banner clears, returns to data

**Status:** COMPLETE
