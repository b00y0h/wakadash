# Milestone v2.0: wakadash

**Status:** SHIPPED 2026-02-19
**Phases:** 4-7
**Total Plans:** 9

## Overview

Full-featured terminal dashboard for WakaTime/Wakapi with async data fetching, visualization panels, and Homebrew distribution. Created standalone wakadash repository to enable homebrew-core submission and add unique value beyond the original wakafetch fork.

**Goal**: Build BubbleTea-based full-screen dashboard with async API calls
**Depends on**: Phase 4
**Plans**: 2 plans

Plans:

- [x] 05-02: Auto-refresh ticker and help overlay

**Details:**

## Milestone Summary

**Key Decisions:**
- Create standalone wakadash repository (enables homebrew-core, adds unique value)
- Use BubbleTea with tea.WithAltScreen() (avoids race conditions vs EnterAltScreen command)
- Self-loop ticker pattern (avoids ticker drift over long dashboard sessions)
- GitHub Linguist colors for languages (familiar colors for developers)
- cenkalti/backoff/v5 for rate limiting (exponential backoff with jitter)
- Personal tap before homebrew-core (immediate distribution while review pending)

**Issues Resolved:**
- Race condition with EnterAltScreen command (solved with tea.WithAltScreen())
- Ticker drift over long sessions (solved with self-loop pattern)
- macOS Gatekeeper warnings (solved with quarantine removal hook)

**Issues Deferred:**
- homebrew-core acceptance (requires >= 30 forks, >= 30 watchers, or >= 75 stars)

**Technical Debt Incurred:**
- None significant
